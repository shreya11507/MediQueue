How this Admin Dashboard Works

  1. Doctor Availability: Pick a doctor and a date (today to +3 days). Click time slots to mark them Unavailable. Save to persist.
  2. Live Queue: Filter by doctor/date/status. Use action buttons to mark In Progress, Completed, Cancelled, or Skipped.
  3. Reports: View totals for selected ranges (today, last 7 days or custom).
  4. Export CSV: Downloads the current filtered queue view.
  5. Integrations: Replace data providers with your Firestore/MySQL API routes.
